# **App Name**: EdTech OS

## Core Features:

- Role-Based Dashboards: Custom dashboards that adapt to the role of the logged-in user, presenting the most relevant information (student, instructor, admin).
- Course Content Studio: A drag-and-drop interface where instructors can create and organize course modules, assignments, and resources.
- AI Writing Assistant: AI assistance that analyzes student writing for argumentation, clarity, and structure, offering real-time iterative feedback and help with citations. It is a tool to make decisions whether the student used information in proper ways or not.
- AI-Powered Research: Enable an intelligent search engine for academic sources that understands natural language queries and automatically generates citations.
- Student Success Hubs: Dedicated portals for international students, local students, and student-athletes, providing resources, checklists, and support.
- Insight & Intervention Hub: Provide predictive analytics that flag at-risk students and suggests personalized interventions to instructors.
- Academic Pathway Simulator: Offer advisors a tool to map out student academic careers, simulating different scenarios to guide long-term planning.

## Style Guidelines:

- Primary color: Deep Indigo (#4B0082) to evoke knowledge, sophistication, and depth.
- Background color: Light Lavender (#E6E6FA) a desaturated tint of Indigo.
- Accent color: Soft Pink (#FF69B4) an analogous color that brings vibrancy and contrasts nicely with indigo.
- Headline font: 'Space Grotesk', sans-serif, for a tech-forward feel; Body font: 'Inter', sans-serif, for readability.
- Use modern, minimalist icons to represent different types of content and actions.
- Maintain a clean, modular layout with clear information hierarchy across all pages.
- Subtle transitions and animations to enhance user experience without being distracting.
